# FIRE RANGER

A 3D visualization of active fires across the world, using data from [NASA's FIRMS](https://firms.modaps.eosdis.nasa.gov/space-apps-2023)

![demo](gif.gif)

## 🛠 Installing

1. Install dependencies

   ```bash
   yarn install
   ```

2. Fire up the server and watch files

   ```bash
   yarn run dev
   ```
